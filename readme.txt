# Know Your City ONA ingester

This Wordpress plugin takes data from the ONA API and puts it into Know Your City's Wordpress website

1. Install under your Plugins directory
2. Copy the files under "theme" to your theme directory and customise
3. Enable in plugin admin
4. Set your ONA username and password
5. Select the forms you want to access
6. Select the settlements you'd like to expose
7. Click "Preview" to see your settlements rendered according to your theme files.
